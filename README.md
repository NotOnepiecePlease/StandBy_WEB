# Projeto WEB do sistema StandBy [Por hora apenas estudos]

### Tecnologias usadas:
- ASP.Net Core MVC 6
- Javascript
- JQuery
- Bootstrap
- Apexcharts
- Cleave.js
- Sweetalert 2

- Chrome (Windows, Mac, Linux)
 - Firefox (Windows, Mac, Linux)
- Safari (Mac)
- Microsoft Edge
- And other WebKit browsers
